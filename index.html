Простое приложение с использованием разных JS-фреймворков

Задача

При открытии страницы делается запрос /core.php?mode=menu на получение меню:

В ответ возвращается json-массив, содержащий поля url и name

[
    {
        "name": "Астрономия",
        "url": "astronomy"
    },
    {
        "name": "Геология",
        "url": "geology"
    },
    {
        "name": "Литературоведение",
        "url": "literatura"
    }
]

После чего происходит отображение полученного меню в формате Twitter Bootstrap: 


<ul class="nav navbar-nav">
  <li><a href="#astronomy">Астрономия</a></li>
  <li><a href="#geology">Геология</a></li>
  <li><a href="#literatura">Литературоведение</a></li>
</ul>

При нажатии на пункт меню он подсвечивается, в области контента появляется анимированная картинка и делается ajax-запрос на сервер: /core.php?mode=content .

Когда получен ответ от сервера, то вместо картинки отображается контент, полученный от сервера. Если произошла ошибка, то картинка тоже пропадает и вместо картинки отображается сообщение «Произошла ошибка».

Пример ответа сервера: 

{
    "name": "Астрономия",
    "url": "astronomy",
    "content": "Контент для пункта астрономия...."
}
